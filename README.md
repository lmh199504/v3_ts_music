# vue3网易云音乐

## 介绍
> 个人练习vue3.0项目 使用Vue3 + TypeScript + Pinia + Vant 实现H5网易云音乐播放播放器
## 在线演示地址
[http://101.35.109.105/home](http://101.35.109.105/home)
## 项目
- 开发框架：Vue3
- 脚手架：Vue-cli
- 组件库：Vant
- 状态管理：Pinia
- 语法扩展：TypeScript、Less
- 打包工具：Webpack
- 代码规范：ESLint

## 实现功能
1. 邮箱登录、手机号登录
2. 首页
3. 歌单详情
4. 跟手详情
5. 歌曲播放
6. 个人中心
7. 用户详情
8. 关注和粉丝
9. 歌曲播放
10. 电台播放
11. 消息发送
12. 歌曲、歌单分享好友
13. 歌词滚动
14. 搜索
15. mv播放
16. 夜间模式
17. 云盘音乐上传
18. 最近播放
19. 歌单创建、编辑
